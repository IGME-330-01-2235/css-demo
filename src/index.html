css-03 : Reset CSS

This branch demonstrates a reset.css stylesheet. There are many different examples of a reset file out there. This particular one came from https://elad2412.github.io/the-new-css-reset/ (yes, even the link styling was reset!). Among all of them, the basic idea is to get rid of all of the default user agent styles so that we're starting from the same baseline in every browser.

Notice how the margin, padding, and font-sizes are gone from every element?
