css-01 : Default Styles

It may not look like much, but there are actually a bunch of different built-in styles that we get by default.

These styles vary a bit from browser to browser. Because of that, they tend to be somewhat ... well ... annoying.

Open the Chrome Inspector (easiest way is to right-click on something and select "Inspect" from the context menu).

On the Elements tab, click around on different elements and see how the styles run.
